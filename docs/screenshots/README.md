## Screenshot Tips

1. **Resolution**: Use 1920×1080 for consistent sizing
2. **Annotations**: Use arrows, highlights, or numbered callouts to draw attention to key elements
3. **Browser chrome**: Close browser tabs, bookmarks bar, and extensions for clean screenshots
4. **Format**: PNG for best quality
5. **Tool recommendations**:
   - **Flameshot**: `flameshot gui` (select area + annotate)
   - **Greenshot**: Region capture with annotation tools
   - **Browser DevTools**: Chrome F12 → Device Toolbar → Capture full size screenshot
